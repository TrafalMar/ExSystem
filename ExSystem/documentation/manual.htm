Начинка

  • При запуску програми ви можете побачити наступне вікно(Вікно проходження опитування)

    У цьому вікні є такі компоненти як кнопки для вибору відповідді, для підтвердження відповідді і для повторного проходження опитування. Також з самого правого боку можете бачити рух по дереву питань.

  • Вікно супервізора

    Весь контент який відображається при запуску програми знаходиться всередині панелі навігації, на цій панелі ви також можете знайти вкладку для адміністрування деревами і експертною системою взагаловм.
